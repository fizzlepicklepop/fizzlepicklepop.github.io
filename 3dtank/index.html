Tiny Tanks Online

A light, cheerful 3D tank duel game with controller support and manual WebRTC multiplayer. Random maps, ricochet shells, first to 5 wins.

How multiplayer works:
1) Host clicks Host and sends the generated string.
2) Friend pastes it and clicks Join.
3) Friend sends the answer string back.
4) Host pastes it and clicks Host: Accept Answer.
Controls
Keyboard: W / S drive, A / D turn, Space fire, Shift brake.
Controller: left stick = drive/turn, A or trigger = fire.
Inspiration: bright toy-like presentation with a clean shopping-channel feel.
